快速入门WRF

# 快速入门教程

本系列教程介绍如何快速入门使用超算集群

本次介绍 WRF 软件

# 步骤1：登陆系统

# 下载软件

下载 SSH 登陆软件，推荐使用 Mobaxterm (opens new window) 软件, 下载并安装即可（可以使用 Portable 版本，双击即可使用）。

# 建立会话

打开软件，新建一个建立 SSH 链接：

  1. 菜单栏 - Sessions - SSH 新建一个回话
  2. 填写远程主机IP Remote Host : 111.198.60.163
  3. 勾选 Specify username，输入用户名；
  4. 修改端口号 Port 为 53103 ， 然后点击 OK 确定；
  5. 然后根据提示，输入登陆密码即可登陆

# 步骤2：加载软件

登陆后，输入 module av 命令，可以查看可用的软件列表。

输入 module add wrf ，即可加载 wrf 软件

# 步骤3：提交作业

用户在本地提交并行的计算任务，可能用过例如 mpirun -n 16 wrf.exe 这样的命令。

在超算集群中，需要用户编写一个提交脚本 sub.sh ，内容写上：

#!/bin/bash
srun -n 16 wrf.exe
1
2

如果不会使用集群上的 vim 编辑器，可以在本地编辑改脚本，然后上传即可。如果出现格式问题，可以执行 dos2unix 脚本名，将脚本改为 Unix 格式。

然后使用 sbatch 命令进行任务提交：

sbatch -n 16 sub.sh
1

然后使用 squeue 命令查看已提交的作业。

squeue
1

# 补充

wrf软件的安装路径为：/work/software/wrf/4.2.2-icc18-impi ，用户可以将其中的例如 run 文件夹拷贝到本地使用，例如

cp -r /work/software/wrf/4.2.2-icc18-impi/run .
1
