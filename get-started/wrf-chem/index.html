# 使用WRF-CHEM进行计算

# 1.REAL

编辑好 namelist.input 以后，chem_opt=0

编写一个提交脚本 Sub_real.sh 内容如下：

#!/bin/bash
srun -n 32 ./real.exe
1
2

在编写提交脚本时，如果不会使用集群上的 vim 编辑器，可以在本地编辑改脚本，然后上传即可。如果出现格式问题，可以执行 dos2unix 脚本名，将脚本改为 Unix 格式。

然后使用命令进行任务提交：

sbatch -n 32 Sub_real.sh
1

# 2.PREP_CHEM

这一部分需要根据用户自己所使用的数据来源进行准备。可以不提供具体的布置。

将wrfbiochemi_、wrfchemi__准备好，放入同文件夹

# 3.REAL

编辑好 namelist.input 以后， chem_opt=你需要的

编写一个提交脚本 Sub_real.sh 内容如下:

#!/bin/bash
ulimit -s unlimited
srun -n 32 ./real.exe
1
2
3

然后使用命令进行任务提交：

sbatch -n 32 Sub_real.sh
1

# 4.WRF

编写一个提交脚本 Sub_wrf.sh 内容如下:

#!/bin/bash
ulimit -s unlimited
srun -n 32 ./wrf.exe
1
2
3

然后使用命令进行任务提交：

sbatch -n 32 Sub_wrf.sh
1

# 数据环境

系统上预先下载好了常用的数据环境，存放在了 /work/software/wrfdata 目录下，用户可以直接使用，或者拷贝到自己的账号下使用，

已配置的常用数据包括：

  • 1.wps地形输入数据——高精度：
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/geog_high_res_mandatory.tar.gz
  • 2.wps地形输入数据——低精度：
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/geog_low_res_mandatory.tar.gz
  • 3.wps地形输入数据——额外数据：
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/geog_nmm.tar.gz
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/geog_thompson28_chem.tar.gz
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/geog_noahmp.tar.gz
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/irrigation.tar.gz
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/geog_ssib.tar.gz
    • https://www2.mmm.ucar.edu/wrf/src/wps_files/lake_depth.tar.bz2
  • 4.prep-chem-sources化学初始数据下载地址：
    • ftp://aftp.fsl.noaa.gov/divisions/taq/emissions_data_2005/em05v2_file*.tar
    • ftp://aftp.fsl.noaa.gov/divisions/taq/emissions_data_2011/em11v1_file*.tar
    • ftp://aftp.fsl.noaa.gov/divisions/taq/global_emissions/global_emissions_v3_24aug2015.tar.gz
  • 5.wrf-chem提供的megan初始数据：
    • https://www.acom.ucar.edu/wrf-chem/megan.data.tar.gz
  • 6.wrf-chem提供的edgar-htap数据：
    • https://www.acom.ucar.edu/webt/wrf-chem/processors/EDGAR-HTAP.tgz
