# 2. 安装扩展包

# 2.1 使用pip安装扩展包

# 配置说明

由于 python 的扩展包数以万计，因此不可能提供一个 完整版 的 python 环境供用户使用。

用户可以根据自身需求，使用系统目录下的 python 命令，或者 共享目录下的 python 命令为基础，使用 conda 或 pip 包管理工具，在用户根目录下自行安装所需要的依赖包即可。

举例如下：

用户需要使用 cartopy, netCDF4, wrf-python, cmaps 这4个包，就可以：

step1 加载系统的 anaconda 环境

module add anaconda3/py38_2021.05
1

step2 使用 pip 命令安装所需的安装包

pip install netCDF4 --user
1

--user 参数表明，安装目录为 ~/.local 而不是共享目录，这样用户就有安装的权限了。

step3 安装后直接使用即可

python
1

# 3. 报错记录

报错：

pip安装报错

解决：

如果遇到安装报错，可以尝试升级 pip 命令到最新版本

python -m pip install --upgrade pip
1

报错：

CommandNotFoundError: Your shell has not been properly configured to use 'conda activate'.

解决：

1 首先终端输入 source activate
2 然后终端输入 source deactivate
3 输入你要激活的虚拟环境指令 conda activate your_virtual_name
1
2
3

报错：

conda版本问题

解决：

升级conda

# 升级conda
conda update -n base -c defaults conda
